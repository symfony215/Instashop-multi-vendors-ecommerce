## requirements
<ul>
                <li>php >= 7.2 </li>
                <li>mysql</li>
                <li>apache</li>
                <li>laravel >= 7.0 </li>
                <li>composer</li>
      </ul>

> you must copy .env.example contents to new file .env ans setup database. 


## installation:

```
composer install

npm install

php artisan cache:clear

php artisan key:generate

php artisan serve

```

> now go to (http//:127.0.0.1:8000)
